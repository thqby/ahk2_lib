# AutoHotkey v2 Learning Path

A structured approach to learning AutoHotkey v2 from beginner to advanced.

## Stage 1: Beginner

**Topics to Learn:**
- Hello World GUI
- Button clicks and events
- Basic array operations
- File reading/writing
- Single hotkey binding
- String concatenation
- Basic math operations

**Recommended Scripts:**
1. **demo_data.ahk** - Array Operations, Map Data Structure

## Stage 2: Intermediate

**Topics to Learn:**
- Multi-pane GUIs with tabs
- Event-driven architecture
- Higher-order functions
- Hotkey context sensitivity
- Class inheritance
- Property getters/setters
- Data structure manipulation

**Recommended Scripts:**
1. **demo_gui.ahk** - GUI Creation, Event Handling
2. **demo_oop.ahk** - Array Operations, Object-Oriented Programming

## Stage 3: Advanced

**Topics to Learn:**
- Multi-window applications
- .NET/COM interop
- Concurrency and async
- Design pattern implementations
- Full MVC applications
- Custom event systems
- Buffer manipulation
- DllCall and WinAPI

## Capstone Projects

After completing the learning path, try building these projects:

1. **Simple GUI Application** - Text editor with file operations
2. **Data Management Tool** - CSV/JSON processor with GUI
3. **Automation Suite** - Hotkey-based workflow automation
4. **System Utility** - Window manager or clipboard enhancer
5. **Advanced Integration** - COM/WinRT application with modern UI
